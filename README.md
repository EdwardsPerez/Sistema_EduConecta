# EduConecta Academic Management System

Academic management system designed for educational institutions. The application helps manage student enrollment and internal records.

## Technologies Used
- ADO.NET
- C#
- SQL Server
- HTML
- CSS

## Features
- Student enrollment management
- Academic record management
- Database integration
- Administrative control panel

## Purpose
This project demonstrates the use of ADO.NET for database management and the development of educational administrative systems.

## Installation

1. Clone the repository
2. Open the project in Visual Studio
3. Configure the database
4. Run the application
